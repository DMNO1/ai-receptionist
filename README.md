# AI Receptionist — 本地服务AI接待员 PoC

> 为本地服务型企业（汽修店、宠物医院、美容院等）提供24/7 AI前台接待

## 产品定位

**一句话价值**：不让任何一个客户电话/消息白打。

**目标用户**：本地服务型小企业（汽修店、宠物医院、餐厅、美容美发、诊所等）

**核心价值**：
- 24/7自动响应客户咨询
- 自动预约登记
- 服务报价查询
- 营业时间/地址导航
- 所有对话留痕可追溯

## PoC功能

| 功能 | 说明 |
|------|------|
| 💬 Web聊天界面 | 客户通过网页与AI接待员对话 |
| 📋 意图识别 | 自动识别预约、询价、问路、问时间等意图 |
| 📅 预约登记 | 收集客户信息并记录预约 |
| 💰 服务报价 | 根据知识库返回服务价格 |
| 🕐 营业信息 | 回答营业时间、地址、电话等 |
| 🤖 LLM增强 | 可选DeepSeek API，无API时使用规则引擎 |

## 快速开始

```bash
cd business/ai-receptionist
pip install -r requirements.txt
py main.py
# 访问 http://localhost:8000
```

## 技术栈

- **后端**: Python FastAPI
- **前端**: HTML + Tailwind CSS + JavaScript
- **对话引擎**: 规则引擎 + 可选LLM增强
- **知识库**: JSON配置（服务项目、价格、营业信息）

## 目录结构

```
ai-receptionist/
├── knowledge_base/
│   └── shop_config.json      # 店铺配置（服务/价格/营业时间）

│   └── chat.js               # 聊天前端逻辑
├── templates/
│   └── index.html            # 聊天界面
├── engine.py                 # 对话引擎（意图识别+回复生成）
├── main.py                   # FastAPI应用
├── requirements.txt
├── .env.example
└── README.md
```
